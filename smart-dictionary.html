smart dictionary

Dutch vocabulary in context, from A2 to C1

WIP

Why I built this

I'm a Dutch learner approaching the B1 level, and I found that existing Dutch-English tools gave translations but not the context I needed to see a word "in action". Knowing that spreken means "to speak" doesn't tell me how to conjugate it, when it's too formal, or what related words I should also know. I built Woordje to be the tool I wished I had.

The Solution

Woordje is a contextual dictionary built for A2/B1 Dutch learners. Instead of bare translations, each entry shows a word in use: example sentences graded by CEFR level, conjugation tables, register labels, and a related-words ecosystem that shows learners how vocabulary connects. I also added click-to-reveal translations to allow learners to test their comprehension before checking.

React.js Vite Supabase SQL Vercel

Key features

  • CEFR-leveled example sentences (A1/A2, B1/B2, C1) with tap-to-reveal English translations, so learners can self-test before checking their understanding
  • Verb conjugation tables covering present, simple past, and perfect tenses
  • Register labels and a related-words ecosystem that group synonyms by formality, helping learners learn similar words for the context
  • Labels to show gender (de/het) on nouns and types of verbs (separable, reflexive) — a persistent pain point for Dutch learners (or at least this one!)
  • Word highlighting across all example sentences so the target word is easy to spot in context

Decisions and tradeoffs

  • I shipped the first version with pre-generated static JSON entries rather than a live database. I chose Supabase so that I could better manage the database with SQL as it grew, and also to practice database skills.
  • I prioritized pedagogically useful entries at the B1/B2 level rather than trying to cover a large word count, under the assumption that 500 well-built entries would serve learners most efficiently.
  • I wrote some example sentences by hand and others with Claude — I deliberately avoided using AI to generate everything to keep difficulty progressive and grammar correct, trading some scalability for quality control.
