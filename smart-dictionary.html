Why I built this

I've been learning Dutch slowly and found that existing tools either don't let me import my own word lists, don't adapt to my weak spots, or require me to maintain two separate apps. I wanted one thing that does all of this, backed by a simple database I can query myself.

The Solution

A personal tool for learning Dutch vocabulary. Instead of static flashcards, this dictionary adapts to what I actually struggle with — surfacing words I've gotten wrong recently, showing example sentences, and tracking progress over time.

Python Flask SQLite Spaced repetition

Key features

  • Word import from CSV
  • Quiz loop that surfaces weak spots using a spaced repetition algorithm
  • Progress tracking over time
